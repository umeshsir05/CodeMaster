Course Overview

C++ is a powerful, high-performance programming language used in system software, game development, and performance-critical applications. This course takes you from C++ basics to advanced concepts including object-oriented programming, templates, and the Standard Template Library (STL).

You'll learn to write efficient, maintainable code while understanding memory management, performance optimization, and modern C++ features (C++11/14/17).

Object-Oriented Programming concepts
Classes, inheritance, polymorphism
Templates and generic programming
Standard Template Library (STL)
Memory management and smart pointers
Exception handling and file I/O
